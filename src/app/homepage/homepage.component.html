Welcome!

Use this Project Management System to manage and monitor your projects. Here are the main features the system can offer:

  • All Projects: List all your projects and track their overall status.
  • Filter Projects: Use different filter options to manage projects. These filtering options include Project Manager, Project Name, Project Topic, Status, Within 1 week, Within 2 weeks, Within 1 mounth and the relevant results will be displayed according to the criteria you select.
  • Create New Project: Create new projects and add them to the management system.
  • Search Project with Id: Access the details of any project with the Project Id and get up-to-date information.
  • Update Project: Update your existing projects, change information such as project name, start date, end date, etc. via Project Id.
  • Delete Project: Remove projects that are no longer needed from the system.

If you have any problems using the platform or need additional assistance, please do not hesitate to contact us. Good work!
